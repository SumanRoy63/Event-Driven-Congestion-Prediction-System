# Event-Driven Congestion Prediction System

## Flipkart Grid 7.0 Hackathon Project

### Features

- Road Closure Prediction
- Event Severity Prediction
- Traffic Hotspot Detection
- Recommendation Engine
- Streamlit Dashboard

### Tech Stack

- Python
- XGBoost
- Random Forest
- DBSCAN
- Streamlit
- Plotly

### Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
